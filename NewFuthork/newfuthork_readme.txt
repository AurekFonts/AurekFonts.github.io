FONT INFORMATION

This font is a canon version of the Star Wars alphabet known as 
"Naboo Futhork". The details of this font were taken from the 
passcodes screen in the 2000 game "Battle For Naboo". 

This font includes all letters, ampersand (&), question mark, and
exclamation point. Note that there is no difference between uppercase
and lowercase letters. Also, kerning has been enabled for this font.
